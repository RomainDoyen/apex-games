Découverte des jeux

Frontend RAWG Dernière mise à jour : 2026-02-15

Objectif

Permettre à l’utilisateur de découvrir des jeux vidéo : page d’accueil avec jeux tendances, catalogue complet avec filtres et pagination, page de détail par jeu.

Composants principaux

  • TrendingGames : affichage des jeux tendances (API RAWG), grille responsive.
  • AllGames : catalogue avec filtres, recherche, pagination.
  • GameCard : carte de jeu (image, titre, métadonnées), lien vers détail et favoris.
  • GameDetails : page détail d’un jeu (id RAWG).

Données

Les listes et détails viennent de l’API RAWG (directement ou via le backend NestJS avec proxy/cache). Clé API : VITE_RAWG_API_KEY (frontend) ou variable côté backend.

UX

  • États de chargement (spinner) et messages d’erreur explicites (clé API manquante, réseau).
  • Design responsive : grille adaptée (desktop / tablette / mobile).

Voir : API, Flux de données.
