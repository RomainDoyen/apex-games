Guidelines d’accessibilité (WCAG 2.1)

WCAG 2.1 Dernière mise à jour : 2026-02-15 Public : tous utilisateurs, daltonisme, basse vision, lecteur d’écran

Principes à respecter

  • Clarté : libellés explicites, hiérarchie de titres cohérente, une action principale par vue si possible.
  • Prévisibilité : mêmes composants et emplacements (header, navigation, cartes jeux), libellés stables.
  • Lisibilité : tailles de texte suffisantes, interlignage confortable, contrastes conformes (AA minimum, AAA visé).
  • Contrôle : animations maîtrisées (pas de clignotement agressif), feedback visuel + texte sur les actions.

Contraste (AAA)

  • Texte normal : ratio ≥ 7:1.
  • Texte large : ratio ≥ 4.5:1.
  • Ne jamais coder une information uniquement par la couleur.

Navigation & focus

  • Tout élément interactif doit être atteignable au clavier (Tab/Shift+Tab).
  • Focus visible : contour net, contraste élevé, non dépendant de la couleur seule.
  • Ordre de tabulation logique (haut→bas, gauche→droite).

Zones tactiles & ergonomie (mobile)

  • Taille interactive minimale : 44×44px.
  • Espacement : au moins 16px entre actions pour éviter les erreurs.
  • Pas de doubles actions ambiguës sur le même écran.

Animations & feedback

  • Animations maîtrisées, sans clignotement excessif (≤ 3 flashes/seconde).
  • Éviter les modales agressives ; préférer des confirmations claires (icône + texte).
  • États de chargement : spinner ou squelette avec texte explicite (“Chargement…”, “Ajout aux favoris”).

Procédure de test (minimum)

Tests visuels

  • Simulation daltonisme (protanopie, deutéranopie, tritanopie).
  • Simulation basse vision (flou, contraste réduit, zoom 200%).
  • Vérification contrastes (Lighthouse + outil de contraste).

Tests d’usage

  • Navigation au clavier (Tab / Shift+Tab) sur tout le parcours.
  • Lecteur d’écran (NVDA sur Windows, VoiceOver macOS, TalkBack sur mobile).
  • Scénarios Apex Games : accueil, catalogue/filtres, détail jeu, favoris, backlog (drag & drop + alternative clavier), connexion/inscription.
